`ide-completions` enables the IDE completions menu system using <kbd>Ctrl</kbd>+<kbd>N</kbd> and <kbd>Ctrl</kbd>+<kbd>P</kbd>:

[![Screenshot](./screenshot.jpg)](./screenshot.jgp)

# Installation

```nushell
use nubrew *
http get https://raw.githubusercontent.com/NotTheDr01ds/ntd-nushell_scripts/refs/heads/main/startup-config/ide-completions/nubrew.nuon | nubrew install
```

Add to your startup config:

```nushell
use ide-completions
```
